# Website filter extension

This is a simple Chrome extension that can put colour filters on web pages (including PDFs).

It has been built using JavaScript.

## Features

- Variety of filters to suit different purposes.
- Easy switching between different filters.
- Works on PDFs.

## Installation

#### 1. clone the repository

```bash
git clone https://github.com/sahars03/Website_filter_extension.git
cd Website_filter_extension
```
#### 2. on Chrome, click the three dots in the top right corner of the window and navigate to 'Extensions'

#### 3. click 'Manage Extensions', and you are taken to the page containing all of your current extensions

#### 4. in the top right corner, switch on the toggle for 'Developer mode'

#### 5. click on the button saying 'Load unpacked' and select the downloaded folder from step 1

#### 6. at the top of the browser, click on the 'Extensions' icon and pin the extension (its logo is an emoji of a computer with books on its screen) so that you can use the filter when the icon is clicked
